Collection of ASL scripts I made for different games, meant to be used as Autosplitters for Livesplit.

The most recent Madagascar Autosplitter currently being used was made by Lox modifying mine to fix some issues.

The Skyrim Special Edition is not used since this version of Skyrim is not played by speedrunners. The script was made as a commision for one runner that wanted to make some tests with it.
